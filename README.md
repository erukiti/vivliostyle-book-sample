# Sample

本のサンプル

## 前提

* [Bun](https://bun.sh/)

別にNode.jsでもいいけどBunが楽

## install

```sh
bun i
```

## 本を作成

```sh
bun run build
```

## プレビュー

```sh
bun run preview
```
